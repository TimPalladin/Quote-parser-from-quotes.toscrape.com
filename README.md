# Парсер цитат с quotes.toscrape.com

### Описание
Простой, но полноценный парсер, который собирает **все цитаты** с сайта (около 100 шт.), сохраняет их в JSON и в красивый Excel-файл.

### Возможности
- Парсинг всех страниц сайта (пагинация)
- Сохранение в `quotes.json`
- Красивый Excel-отчёт с форматированием
- Обработка тегов, подсчёт количества тегов
- Защита от дублей

### Технологии
- Python
- requests + BeautifulSoup4
- pandas
- openpyxl
